# Stack

### Discussion about Stacks

#### Definition
One-ended linear data structure.

#### Use senario
- Undo mechanisms
- Compiler syntax checking
- Model a pile
- Keep tracking previous behaviors
- DFS

#### Complexity
| Action      | Complexity|
| ----------- |------|
| Pushing     | O(1) |
| Popping     | O(1) |
| Peeking     | O(1) |
| Searching   | O(n) |
| Size        | O(1) |
