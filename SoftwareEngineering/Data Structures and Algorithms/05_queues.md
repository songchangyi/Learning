# Queues

### Definition
A linear data structure models real world queues.

### Terminology
- Enqueue
- Dequeue

### When and where
- Any waiting line models
- Keep track of the x most recently added elements
- First come first serve
- BFS

### Complexity Analysis

| Actions | Complexity |
| --- | --- |
| Enqueue | O(1) |
| Dequeue | O(1) |
| Peeking | O(1) |
| Contains | O(n) |
| Removal | O(n) |
| Is Empty | O(1) |
